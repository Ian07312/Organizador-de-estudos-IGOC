Organizador (Cronograma) de Estudos IGOC
O problema encontrado a ser solucionado é uma ajuda para estudantes a fim de organizar os seus estudos, seja para Ensino Médio, cursos ou Ensino Superior. 
A solução encontrada foi uam interface Gráfica, onde o aluno identifica a matéria, o data e horário da prova e seu conteúdo, o Cronograma grava as informações conforme a dificuldade do aluno naquela matéria.
O público-alvo são os estudantes que buscam uma forma de organizar os seus estudos
As funcionalidades do sistema contém o Cadastro de Matérias, um Cronômetro de estudos, Sistema de Prioridade, Controle de tempo por matéria (aluno ver qual foi a que mais estudou), Status da matéria e um sistema de revisão.
Base do sistema: Electron, HTML na estrutura da interface, CSS na estilização completa do sistema, Javascript é o todo do sistema, Node.js dentro no Electron e Charts.js.
Para instalar o sistema, é necessário ter o Node.js instalado, baixar o projeto pelo GitHub, acessar a pasta do projeto, instalar as dependências com npm e executar o aplicativo, que abrirá como um programa desktop.
Para executar o sistema, basta acessar a pasta do projeto no terminal, instalar as dependências com o npm e iniciar o aplicativo com o comando de execução, que abrirá automaticamente como um programa desktop. 
Para rodar os testes do sistema, é necessário ter o Node.js instalado, acessar a pasta do projeto, instalar as dependências com npm e executar o comando de testes. Após isso, o sistema executará automaticamente os testes implementados com Jest, exibindo no terminal os resultados de sucesso ou erro.
Para executá-lo, basta acessar a pasta do projeto e rodar o comando de lint, que irá verificar automaticamente possíveis erros, inconsistências e padrões de código, exibindo os resultados no terminal.
A versão atual do sistema é 1.0.0.
