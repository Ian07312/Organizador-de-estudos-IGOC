# Organizador (Cronograma) de Estudos IGOC

![CI](https://github.com/Ian07312/Organizador-de-estudos-IGOC/actions/workflows/ci.yml/badge.svg)

---

## Descrição do Problema

O problema encontrado a ser solucionado é a dificuldade que estudantes possuem em organizar seus estudos, seja para Ensino Médio, cursos técnicos ou Ensino Superior. A falta de organização pode impactar diretamente no desempenho acadêmico e na produtividade.

---

## Proposta de Solução

A solução desenvolvida consiste em uma interface gráfica onde o aluno pode cadastrar matérias, inserir data e horário de provas, além de registrar conteúdos. O sistema organiza automaticamente essas informações, considerando o nível de dificuldade definido pelo aluno.

---

## Público-alvo

Estudantes que buscam uma forma eficiente de organizar seus estudos e melhorar seu desempenho acadêmico.

---

## Funcionalidades

- Cadastro de matérias  
- Cronômetro de estudos  
- Sistema de prioridade (Alta, Média, Baixa)  
- Controle de tempo por matéria  
- Status da matéria (Pendente/Concluído)  
- Sistema de revisão de conteúdos  

---

## Instalação

Para instalar o sistema, é necessário:

1. Ter o Node.js instalado
2. Ver se o node e o npm estão instalados  
3. Baixar o projeto pelo GitHub  

---

## Execução

Para executar o sistema:

1. Acesse a pasta do projeto no terminal  
2. Instale as dependências com npm   
3. Execute o comando de inicialização 
4. Instalar as dependências (npm install)
5. Após as instalações de o comando: npm start

---

## Testes

Para rodar os testes do sistema:

1. Acesse a pasta do projeto  
2. Instale as dependências com npm  
3. Execute o comando de testes  

Os testes automatizados com Jest serão executados, exibindo no terminal os resultados.

---

## Lint

Para executar o lint:

Acesse a pasta do projeto e execute o comando de lint. O sistema analisará automaticamente o código, verificando possíveis erros, inconsistências e padrões.
